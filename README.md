# lsdu
Hybrid of ls, du and df

```
lsdu [OPTION]... [FILE|DIRECTORY]...
  -apparent
        show apparent size
  -free
        report free space
  -human
        display size in KiB, MiB etc
  -k    display size in KiB
  -m    display size in MiB
  -sort
        sort by size (default true)
  -total
        report a total for all files
```
